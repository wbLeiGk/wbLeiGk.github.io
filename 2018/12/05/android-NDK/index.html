Java 使用dll

c语言头文件生成

  • 使用javac javaName.java 生成.class 文件
  • 使用javah javaName 生成 .h 头文件

note: 倘如您的java中有包名，那么退回到包名的根目录，使用以下形式的命令

  • javac com\application\javaName.java 生成.class 文件
  • avah com.application.javaName 生成 .h 头文件
